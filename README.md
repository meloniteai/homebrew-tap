# Melonite Homebrew Tap

Homebrew formulae for [Melonite](https://melonite.ai). Updated automatically by
CI on each release.

## Usage

```sh
brew install meloniteai/tap/melonite
```

That's shorthand for tapping this repository and installing the formula:

```sh
brew tap meloniteai/tap
brew install melonite
```

Works on macOS and Linux (amd64/arm64).

## Available formulae

| Formula | Description |
|---|---|
| `melonite` | The Melonite client — a headless MCP server for agent harnesses |

## Upgrading

```sh
brew update
brew upgrade melonite
```
